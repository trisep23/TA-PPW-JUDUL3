Landing page responsif untuk **perusahaan teknologi** yang menyediakan jasa pembuatan website profesional.  
Dibangun menggunakan **Tailwind CSS** untuk tampilan modern, bersih, dan responsif.

## Tujuan Proyek
Proyek ini merupakan **Tugas Akhir Praktikum Pemrograman Web Judul 3**.  
Tujuannya adalah untuk menerapkan framework **Tailwind CSS** dalam pembuatan website yang responsif, elegan, dan user-friendly.

---

## Struktur Halaman
Landing page ini terdiri dari enam komponen utama sesuai dengan ketentuan tugas:

1. **Header / Navigation** – Logo dan menu navigasi yang responsif.  
2. **Hero Section** – Headline utama, deskripsi singkat, dan tombol Call To Action (CTA).  
3. **Features Section** – Tiga fitur utama layanan dengan ikon dan deskripsi.  
4. **About Section** – Informasi singkat mengenai perusahaan WebStart Studio.  
5. **Contact Form** – Formulir kontak dengan field *Nama*, *Email*, dan *Pesan*.  
6. **Footer** – Hak cipta dan tautan media sosial (Instagram, LinkedIn, GitHub).

---

## Teknologi yang Digunakan
- **HTML5**
- **Tailwind CSS (via CDN)**
- **Remix Icon** (untuk ikon)
- Tidak menggunakan framework JavaScript tambahan.
